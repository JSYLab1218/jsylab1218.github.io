# Jye-Shane Yang Laboratory website

A ready-to-deploy static website for GitHub Pages.

## Files

- `index.html` — homepage
- `professor.html`
- `research.html`
- `members.html`
- `publications.html`
- `photographs.html`
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/images/` — placeholder images

## Publish on GitHub Pages

1. Upload all files and folders to the repository root.
2. Open **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select **main** and **/(root)**.
5. Save.

## Replace images

Replace the SVG placeholder files with your own JPG/PNG files and update the
`src` paths in the HTML. Avoid spaces and Chinese characters in file names.

## Important

The example email address and some personal/member/publication entries are
placeholders. Verify all details before publishing.
